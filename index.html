Import Past Season Data

Three ways to load 2024 / 2025 historical records into PoTa Tracker:

Option 1 — JSON Backup
Click Import Backup (above) and select a previous pota-backup-*.json file to restore all records instantly.
Option 2 — Pull from Google Sheets
Enter past rows directly into your Google Sheet (same column headers as the GAS schema), then click Pull All Data in the Sheets Integration card.
Option 3 — Manual Entry with Past Dates
Use the Plantation / Yield / Pest forms and simply set the date field to a past date (e.g. 2024-03-15). The Comparative Analysis charts automatically group data by year and season.
Option 4 — importBatch API (Advanced)
The v4 GAS script accepts bulk imports: POST to your GAS URL with
{ "action":"importBatch", "type":"plantation", "records":[{...}] }
Valid types: plantation, grower, yieldAssessment, pestDisease.
All data auto-saved to localStorage. Use Export/Import for backup.
